# E-Commerce Frontend Project

## Fitur
- Halaman Beranda dengan banner
- Halaman Daftar Produk (10+ produk)
- Pencarian & Filter produk
- Halaman Detail Produk
- Keranjang Belanja (Cart)
- Checkout dengan validasi form
- Design Responsif (mobile & desktop)
- Routing dengan React Router

## Teknologi
- React 18
- Vite
- React Router DOM 6
- React Icons
- CSS Modules
